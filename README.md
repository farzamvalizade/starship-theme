# Installation
```bash
sudo pacman -S starship
```
You can use other package manager e.g., APT/ZIPPER/...
then run this command 
```bash
echo 'eval "$(starship init zsh)"' >> ~/.zshrc
```
you can change the .zshrc and zsh to bash,fish and etc.
# Usage
Put the `starship.toml` in this path:
`~/.config/starship.toml`

Then Enjoy :)
